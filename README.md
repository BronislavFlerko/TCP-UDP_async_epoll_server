# TCP/UDP Async Server
### Задание выполнил Бронислав Флерко

Асинхронный сервер на C++ для обработки TCP и UDP подключений с использованием epoll.

## Технологии:
C++17, Linux sockets, epoll

## Особенности

- Асинхронная обработка подключений через epoll
- Поддержка TCP и UDP протоколов
- Мультиклиентская архитектура
- Командный интерфейс

## Команды

- `/time` - текущее время и дата
- `/stats` - статистика подключений  
- `/shutdown` - завершение работы сервера

## Сборка

```bash
mkdir build && cd build
cmake ..
```

## Запуск

```bash
./TCP-UDP_epoll_server
```

Сервер запускается на порту 8888.

## Подключение
```bash
#Локально
#TCP
nc localhost 8888

#UDP  
nc -u localhost 8888  
  
#С другого устройства (ваш ip)

# TCP подключение
nc 192.168.1.100 8888

# UDP подключение  
nc -u 192.168.1.100 8888
```

## Формат работы

Если сообщение клиента начинается с символа /, то оно интерпретируется как команда.
 В противном случае зеркалируется клиенту.
